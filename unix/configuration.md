# Configuration

## Configuration

1. To extend `PATH` variable use

    ```bash
    PATH="${PATH}:new_path"
    ```

1. To add ssh-key use

    ```bash
    ssh-add path_to_ssh_key
    ```

1. Aliases
    * **`alias`** - list all aliases
    * **`alias bb='echo bb'`** - create an alias
    * **`unalias bb`** - remove an alias
